# AI-Driven Growth: Insights into Revenue Trends and Stock Reactions

## Objective
Capstone project analyzing the relationship between AI revenue 
growth and stock market reactions across major tech companies 
(2015-2024).

## Research Questions
1. How does AI revenue growth relate to stock price volatility?
2. How do event types affect stock performance?
3. Which periods had extreme stock movements?

## Tools & Libraries
- Python (pandas, seaborn, matplotlib, numpy)
- Jupyter Notebook

## Key Findings
- GPT-4 release triggered the highest single stock impact event
- Companies with higher AI revenue growth showed greater 
  stock volatility
- Revenue growth trends vary significantly by company, 
  with clear acceleration post-2022

## Dataset
- 10,959 records covering AI revenue and stock impact data
- Event-level analysis on 233 cleaned records

## How to Run
1. Clone this repo
2. Open `Insights into Revenue Trends and Stock Reactions by Amira Hanis.ipynb` in Jupyter Notebook
3. Run all cells sequentially
